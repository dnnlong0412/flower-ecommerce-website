# Flower E-commerce Website

## Description
A web-based flower e-commerce application built with PHP and MySQL using XAMPP.

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## Features
- Product management
- Shopping cart
- Order processing
- Admin dashboard

## How to run
1. Place project in htdocs folder
2. Start Apache and MySQL in XAMPP
3. Import database into phpMyAdmin
4. Access via localhost
